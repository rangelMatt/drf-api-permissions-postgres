### Setup

To deploy locally:

- `python3 -m venv .venv`
- `source .venv/bin/activate`
- `pip install -r requirements.txt`
- `python manage.py runserver`
- `docker compose run`

### Tests

To initiate tests, run `python3 manage.py test`# django-models
